# สายโจรจำลอง (Scam Call Trainer)

Practice taking a scam call before the real one comes: an AI plays the scammer, you spot the red flags and hang up in time. Scored, shareable, grounded in official AOC 1441 / police scam statistics and case-type taxonomy.

**Status:** planned — idea + data verified, spec pending. See [`docs/STARTER.md`](docs/STARTER.md) for the project brief.
